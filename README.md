# Parser-NFT-gifts-EN-RU-
EN: Generates Telegram NFT gift links from 103 collections. Two modes: 10 random or one of each type. Links are saved to links.txt. 

RU: Программа генерирует ссылки на NFT-подарки Telegram из 103 коллекций. Доступно 2 режима: 10 случайных или по одному каждого типа. Ссылки сохраняются в links.txt.
