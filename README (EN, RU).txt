(EN)

1. Launch via Application (.exe)

Open the dist folder and select the version in your preferred language (the language is indicated in the folder and file names). Launch the application.
After clicking one of the buttons in the program, a text document containing the generated gift links will be automatically created in the same folder where the application is located.

Open this text document to access and use your NFT gift links.

2. Launch via Python File (.py)

Select the required file (the language is indicated in the file name). Open it using any preferred IDE (such as Visual Studio Code or PyCharm), the standard Python interpreter,
or the command line (cmd). Once launched and an action is performed, a text document will be created in the same folder as the script, storing the generated links.

Open this document to view and use the links to the NFT gifts.



(RU)

1. Запуск через приложение (.exe)

Откройте папку dist и выберите версию на нужном вам языке (язык указан в названии папок и файлов). Запустите приложение. После нажатия одной из кнопок
в программе в той же папке, где находитсяисполняемый файл, автоматически создастся текстовый документ со сгенерированными ссылками на подарки.

Откройте этот текстовый документ, чтобы воспользоваться полученными ссылками на NFT-подарки.

2. Запуск через Python-файл (.py)

Выберите нужный вам файл (язык указан в названии). Откройте его с помощью любой удобной IDE (Visual Studio Code, PyCharm и др.), через стандартный интерпретатор
Python или командную строку (cmd). После запуска и выполнения команды в папке с файлом создастся текстовый документ, в котором будут сохранены сгенерированные ссылки.

Откройте этот документ для доступа к ссылкам на NFT-подарки.
